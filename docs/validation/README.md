# Validation Results

This directory is reserved for real-world Phase 2 validation artifacts.

Planned coverage:
- 5 to 10 SimJEB bracket models
- 3 to 5 GrabCAD bracket models
- one report bundle per model
- side-by-side notes on whether SimReady findings match expected geometry complexity

Suggested contents per model:
- original STEP file reference
- terminal report snapshot
- JSON report
- HTML report
- short manual verdict

Recommended naming:
- `simjeb_<model-name>.json`
- `simjeb_<model-name>.html`
- `grabcad_<model-name>.json`
- `grabcad_<model-name>.html`

Do not commit proprietary source CAD unless sharing is allowed.
